# Section-1
## 目录结构
```
├── README.md - 说明文档
├── assets - 编译后的静态文件
├── client - 客户端源代码
│   ├── actions - 操作目录
│   ├── app.less - app 自定义的样式文件
│   ├── components - 组件目录
│   ├── constants - 常量目录
│   ├── index.js - 主文件
│   ├── services - 服务目录
│   └── stores - 数据目录
├── index.js - 服务端主文件
├── package.json - 项目定义
└── server - 服务端源代码
    ├── apis - api目录
    ├── index.js - 主文件
    └── models - model目录
```
